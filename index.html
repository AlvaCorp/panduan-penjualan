Panduan Aplikasi Toko Obat

Untuk mulai menggunakan Aplikasi Toko Obat, silakan melakukan login:

Login Aplikasi

  1. Buka aplikasi web browser (Mozilla Firefox/Google Chrome)
  2. Masukkan URL pada address bar : http://localhost untuk membuka halaman Login
  3. Masukkan Username dan Password
  4. Klik tombol Login
  5. Jika Username dan Password cocok dengan User yang terdaftar pada sistem, otomatis masuk ke dalam halaman Aplikasi Toko Obat

Silakan klik Panduan Kasir atau Panduan Admin.

Mengganti Password Pengguna

  1. Saat sedang login pada Aplkasi, klik link Ganti Password di kanan atas.
  2. Masukkan Password Lama
  3. Masukkan Password Baru
  4. Ulangi Password Baru
  5. Klik tombol Ganti Password
