Manajemen Produk

Adalah menu/fitur untuk melakukan pengaturan Produk yaitu:

  • Daftar dan Pencarian Produk
  • Input Produk Baru
  • Lihat Detail Produk
  • Edit Produk
  • Cetak Daftar Harga

Daftar dan Pencarian Produk

Adalah Daftar Produk yang terdata pada sistem.
Akses melalui menu Produk –> Cari Produk

Keterangan Gambar:

  1. Tombol Input Produk Baru untuk menambah Produk
  2. Kolom Pencarian Produk
  3. Daftar Produk
  4. Tombol Detail Produk untuk melihat detail produk
  5. Tombol x untuk menghapus produk (tombol ini muncul jika stok produk kosong)
  6. Daftar Halaman adalah link menuju daftar produk pada halaman berikutnya (setiap halaman terdapat 25 daftar Produk)

Input Produk

  1. Klik tombol Input Produk Baru
  2. Isikan Nama Produk, Deskripsi, pilih Satuan Produk, jumlah Stok, Harga Modal, dan Harga Jual
  3. Klik tombol Input Produk

Detail Produk


Pada halaman ini, Admin dapat melakukan:

  1. Tambah Satuan Produk
  2. Edit Stok dan Harga
  3. Konversi Satuan Produk
  4. Mutasi Stok
  5. Edit Nama dan Deskripsi Produk

Tambah Satuan

  1. Lihat bagian Konversi Satuan
  2. Pada Tambah Satuan Baru, pilih Satuan yang ingin ditambahkan
  3. Klik tombol Tambah

Edit Stok dan Harga

  1. Lihat bagian Stok dan Harga
  2. Ubah Stok sesuai Satuan yang diinginkan
  3. Ubah Harga Modal dan Harga Jual
  4. Klik tombol Update untuk menyimpan perubahan

Konversi Satuan

Adalah fungsi untuk mengatur Satuan besar dan kecil produk dengan jumlah satuannya.
Misalnya:

  • Produk memiliki satuan Dus dan Botol
  • Satu Dus = 10 Botol

Pengaturan Konversi Satuan:

  1. Isikan Angka konversi dan pilih Satuan Kecil-nya
  2. Klik tombol Update

Mutasi Stok

Mirip seperti pengoperasian Mutasi Stok pada Kasir, untuk Admin mutasi stok dapat dilakukan pada halaman Detail Produk :

  1. Pada bagian Mutasi Stok
  2. Pilih Dari Satuan
  3. Isikan angka Sejumlah
  4. Pilih Ke Satuan
  5. Klik tombol Mutasikan

Edit dan Hapus Produk

Untuk mengubah Nama dan Deskripsi produk:

  1. Pada halaman Detail Produk, klik tombol Edit Produk
  2. Ubah Nama dan Deskripsi produk
  3. Klik tombol Update Data Produk untuk menyimpan perubahan
  4. Tombol Hapus Produk ini untuk menghapus Produk dari sistem

Catatan:

Penghapusan Produk dari sistem dapat dilakukan jika Stok produk telah kosong.

Cetak Daftar Harga

Untuk mencetak daftar harga produk terbaru

  1. Buka dengan menu Produk –> Cetak Daftar Harga
  2. Klik tombol Cetak untuk membuka kotak dialog Cetak/Print
